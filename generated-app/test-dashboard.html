🧪 Authentication Flow Testing Checklist

  • Sign Up Flow: Create new account → Auto sign-in → Redirect to dashboard
  • Sign In Flow: Email/password → Successful login → Dashboard access
  • Google OAuth: Google sign-in button → OAuth flow (requires valid credentials)
  • Wallet Connect: Wallet button → Modal → Demo wallet selection → Authentication
  • Profile Management: View/edit user profile → Save changes → Success confirmation
  • Session Management: Sign out → Redirect to home → Protected routes require auth
  • Error Handling: Invalid credentials → Error message → Retry option

🎨 Theme Consistency Testing Checklist

  • Theme Toggle: All pages have theme toggle in consistent location
  • Background Colors: Page backgrounds change properly in light/dark mode
  • Text Contrast: All text is readable in both themes
  • Component Theming: Cards, buttons, inputs adapt to theme
  • Modal Theming: All modals have non-transparent, theme-aware backgrounds
  • Navigation Theming: Headers, sidebars, bottom nav follow theme
  • Icon Consistency: Icons and graphics adapt to theme appropriately
  • Theme Persistence: Theme choice persists across page navigation
